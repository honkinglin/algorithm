# 1385. Find the Distance Value Between Two Arrays <Badge type="tip" text="Easy" />

Given two integer arrays `arr1` and `arr2`, and the integer `d`, *return the distance value between the two arrays*.

The distance value is defined as the number of elements `arr1[i]` such that there is not any element `arr2[j]` where `|arr1[i]-arr2[j]| <= d`.

> Example 1:  
Input: arr1 = [4,5,8], arr2 = [10,9,1,8], d = 2   
Output: 2   
Explanation:   
For arr1[0]=4 we have:   
|4-10|=6 > d=2    
|4-9|=5 > d=2    
|4-1|=3 > d=2    
|4-8|=4 > d=2    
Thus 4 satisfies the distance requirement.  
For arr1[1]=5 we have:   
|5-10|=5 > d=2    
|5-9|=4 > d=2    
|5-1|=4 > d=2    
|5-8|=3 > d=2   
Thus 5 also satisfies the distance requirement.   
For arr1[2]=8 we have:   
|8-10|=2 <= d=2   
|8-9|=1 <= d=2   
|8-1|=7 > d=2   
|8-8|=0 <= d=2   
It has distances <= 2, thus it doesn't satisfy the condition.    
Therefore only arr1[0]=4 and arr1[1]=5 satisfy the distance condition, yielding a distance value of 2.

> Example 2:  
Input: arr1 = [1,4,2,3], arr2 = [-4,-3,6,10,20,30], d = 3  
Output: 2

> Example 3:  
Input: arr1 = [2,1,100,3], arr2 = [-5,-2,10,-3,7], d = 6   
Output: 1

var findTheDistanceValue = function(arr1, arr2, d) {
    function inRange(nums, low, high) {
        let left = 0;
        let right = nums.length - 1;

        while (left <= right) {
            const mid = Math.floor((left + right) / 2);

            if (nums[mid] >= low && nums[mid] <= high) {
                return true;
            }

            if (nums[mid] < low) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return false;
    }

    arr2.sort((a, b) => a - b);
    let count = 0;

    for (let x of arr1) {
        if (!inRange(arr2, x - d, x + d)) {
            count ++;
        }
    }

    return count;
};
